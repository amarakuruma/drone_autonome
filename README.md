# Drone Controle 1
premiere V 
## Description
c'est un programe qui permet de gerer un drone tout en affichant l'etat du drone en vol ou au sol 
## Fonctionnalités
nous avons cree un menu interactif avec l'utilisateur et aussi nous avons configurer plusiurs etats comme decoller atterir envol
## Comment lancer le projet
python drone.py
uvicorn api:app --reload --port 8001
## Technologies utilisées
FASTAPI
UVICORN
PYTHON
